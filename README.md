With this software you can build a very smart led light, complete with voice control.

### application example
I printed a tube, winded a 3mm wide cob ledstrip around it and have the fet module and esp inside. 

### main features
- easy to connect to wifi via espressif rainmaker app
- controllable via web ui, google-home, optional button or http
- 4 advanced timers (a.o. configurable dim level)
